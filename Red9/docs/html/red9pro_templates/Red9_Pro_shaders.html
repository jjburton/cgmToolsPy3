Pro_Pack : shaders¶

ProPack shaders module contains useful tools for managing shaders in Maya

>>> # import statement for the module via the r9pro decompiler
>>> from Red9.pro_pack import r9pro
>>> r9shaders = r9pro.r9import('r9shaders')

This module covers simple base shader functions

get_shaders(node)¶

return the shader / shaders for given node

copy_materials(source, target)¶

copy shaders between the 2 given meshes. If the source has shaders assigned to faces then these are also copied over if possible

Parameters:
  • source – source mesh (transform) assigned to the shaders we want to copy
  • target – target mesh (transform) to copy the shaders over too
sync_uv_sets(source, target, delete_history=False)¶

sync the uv_sets between the source and the target geo’s. This will add uv_sets that don’t exists in the target but are in the source, and delete any that are in the target but not in the source. We also clean up any createUVset or deleteUVset nodes if the delete_history flag is set : simple delete(node, ch=True) call so be aware of that

Note

this does NOT update the actual uv’s, we’re only dealing with the uv_set management here

Parameters:
  • source – source mesh (transform) containing the uv_sets we want to sync to the target
  • target – target mesh (transform) the one we want to create / delete uvSets too
