create_tracked_path(time=(), track_pnt=None, step=10, name='', rebuild=True, spans=10)¶

create the visual path reference which we also use for the wire deformer

Parameters:
  • time – the time we’re going to cover (start, end)
  • track_pnt – the node we’re going to track to create the curve
  • step – the frame step between knots, only if rebuild=False
  • name – name of the curve we’re going to create
  • rebuild – if True we create the curve with a linear knot per frame, then we reduce the curve using the rebuild codebase in Maya. If False then we create the curve uniformly with a knot at each frm as governed by the frame “step” input
  • spans – the target span count if using the rebuild method
