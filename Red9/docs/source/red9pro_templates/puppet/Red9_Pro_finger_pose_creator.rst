Pro_Pack : puppet : finger_pose_creator
================================

ProPack puppet call to push a library of finger poses over to an existing PuppetRig system as
finger sliders. The poses must be made using the "FINGERMIRRORED" pose handler in the Red9 PoseUI.
There's currently no UI support for this so please read carefully

	>>> # import statement for the module via the r9pro decompiler
	>>> from Red9.pro_pack import r9pro 
	>>> r9puppet_fingerposes = r9pro.r9import('r9puppet_fingerposes')
